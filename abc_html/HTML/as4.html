PROGRAMMING LANGUAGES

A programming language is a formal language designed to communicate instructions to a machine, particularly a computer. Programming languages can be used to create programs that control the behavior of a machine and yo express algorithms precisely.

C++

C++ is a programming language that is general purpose, statically typed, free-form and compiled. It is regarded as an intermediate level language, as it comprises both high level and low level language features.

C

C is one of the most widely used programming languages of all time, and c compilers are available for the majority of available computer architectures and operating systems.

Java

Java is a computer programming language that is concurrent, class-based, objec oriented and specifically designed to have as few implementation dependencies as possible. It is intended to let developers "write once, run anywhere" (WORA), meaning that code that runs on one platform does not need to be recompiled to run on another.

O2 and life

Oxygen(O2) is one of the most important elements required to sustain life. Without it, our health begins to suffer and/or we die. Unhealthy or weak cells due to improper metabolism lower their natural immunity.
